As credenciais do banco devem ser informadas no arquivo
resources/credentials.xml com o seguinte formato:


```
<?xml version="1.0"?>
<credentials>
  <user id="1001">
    <hostname></hostname>
    <dbname></dbname>
    <login></login>
    <password></password>
  </user>
</credentials>
```
